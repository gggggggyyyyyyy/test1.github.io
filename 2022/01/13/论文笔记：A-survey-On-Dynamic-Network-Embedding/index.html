论文笔记： A Survey on Dynamic Network Embedding

论文出处： Machine Learning 2020

Abstract

现实世界的网络由各种各样的交互的演化信息，而现有的研究仅仅简单将其描述为静态的网络，没有考虑动态网络上的演化趋势。动态图嵌入在保留网络结构和时序信息的前提下，将节点表示为低维稠密的向量。

根据动态图嵌入基于的技术，对动态图嵌入方法进行划分：

  1. 基于矩阵分解的动态图嵌入
  2. 基于skip-gram的动态图嵌入
  3. 基于自编码器的动态图嵌入
  4. 基于神经网络的动态图嵌入
  5. 其他动态图嵌入方式

现有算法面临的挑战：如动态嵌入模型、大规模动态网络、异构动态网络、动态属性网络、面向任务的动态网络嵌入以及更多的嵌入空间。

Introduction

图结构网络是复杂系统常用的载体，eg: PPI、SN、语义网络、犯罪网络。由于现实世界中的网络是不断演化的，单纯的利用静态网络进行嵌入表示会增加时空开销，因为一旦网络变化就需要重新进行训练。

  1. DeepWalk：将静态网络嵌入作为序列建模问题。该方法通过随机游走获取sequence，然后将其喂入skip-gram来预测输入节点的上下文。
  2. node2vec：基于DeepWalk发展而来，nodec2vec将DFS、BFS的游走策略进行组合，来更加灵活的学习网络结构。
  3. LINE: 为了考虑一阶邻居节点和二阶邻居节点信息，设计了特定的loss函数，该方法适用于大规模网络的表示学习。
  4. SDNE: 深度的网络结构图嵌入模型(A Structural Deep Graph Embedding Model),将一阶邻居信息与二阶邻居信息的优势都利用起来，然后还保持了结构的高阶非线性。
  5. DNE-APP: 提供了深度网络嵌入模型。采用半监督的方式，通过堆叠自编码器来得到紧凑的表示学习。
  6. ANE: 利用对偶自编码器进行变分推测，将节点的嵌入后验与先验匹配生成网络表示。
  7. APNE: 矩阵分解的方式融合结构和节点标签信息。

静态网络缺乏了描述网络演化的能力。节点的新增和删除与边的新增和删除都在静态网络上不能进行表示。动态图涉及了历史信息、图快照、时间戳、节点和边的定义。动态网络根据数据的处理方式分为两类：（一)图快照（二）连续时间网络

根据图快照之间的时间间隔，可以分为等长时间序列快照、不等长时间序列快照。但是将动态图切割为等长时间间隔的图快照，仅仅能粗糙的描述其整体变化，并不能保留其详细的网络演化信息，而连续时间网络可以弥补这一缺点，每个标注边上有时间戳信息来预测节点之间刚刚是否产生了连边，每个边都有特定的时间戳。

Zhou设计了一种动态三元组(Dynamic Triad)模拟三元组如何从开三角变成闭三角，该方法保留了演化网络的结构和动态信息。
Nguen将时间依赖性引入节点嵌入和深度图模型中，从而整合更多时间信息。

从动态网络嵌入模型和预处理过程两个方面构建了动态网络嵌入内容体系。具体来说，需要将具有属性和标签的网络数据集预处理成统一的线性数组，然后结合时间序列进行网络动态演化生成归一化矩阵。然后，通过优化算法更新归一化矩阵来构造网络表示。

标记

分类

动态网络是表示为快照还是用时间戳标记的连续时间网络。根据该策略，我们将动态网络嵌入分为五大类:基于特征值分解的嵌入、基于跳跃图的嵌入、基于自动编码器的嵌入、基于图卷积的嵌入等。

基于矩阵分解的动态图嵌入

传统但有效的嵌入方法是在复杂网络中通过奇异值分解对邻接矩阵进行分解，通过特征分解对属性矩阵进行分解，从而构造每个节点的嵌入表示。从矩阵分解的角度来看，网络的动态演化相当于原始矩阵的不断变化，其中网络嵌入根据矩阵的摄动理论进行更新。


动态网络嵌入中特征值分解的一个实例。在快照t时，模型通过与网络结构A的结合产生频谱嵌入结果，得到静态嵌入结果Y。之后，在快照t + 1处，拓扑变化∆A表示网络结构的动态信息。该模型结合矩阵摄动理论，对新的嵌入Y T进行更新。

TIMERS最优地设置SVD的重启时间，以减少时间上的错误积累。

基于skip-gram的图嵌入

Word2Vec的广泛使用促进了著名的Skip-Gram模型，通过该模型，我们可以从输入中预测上下文。

  1. node2vec 有偏的随机游走
  2. LINE 构造Loss、一阶相似、concate、二阶相似
  3. 在LINE基础上进行构造可分解的目标函数应用于动态图，(仅仅部分节点同时迭代更新) –多标签分类任务
  4. 无监督网络表示 (基于random walk)

静态网络上的嵌入通常通过random walk获得训练的语料库和文集，然后将这些语料库和文集输入到skip-gram中进行训练。但是random walk没有考虑连边的产生的先后顺序。例如：消息传播通常是有向的，但是无限制的随机游走可能得到完全相反顺序的语料库和文集。基于该问题，Nguyen将动态网络表示为连续时间网络的形式，每条连边上存在时间戳，在进行随机游走的时候，必须依照连边上的时间顺序进行游走，以上游走规则使得该方法优于DeepWalk和node2vec。
Dynnode2vec： 改进node2vec,通过捕获t-1时刻的信息更新t时刻的node2vec。该方法仅仅考虑了变化的节点，而不是snapshot内的全部节点，局部区域变化的random walk可以有效的更新网络演化中的嵌入向量。
HTNE： 利用Hawkes过程捕捉历史邻域对当前邻域形成序列的影响。

基于自编码器的图嵌入

自编码器是一种人工神经网络，是一种无监督的方式对数据进行降维。自编码器可分为编码器、解码器，自编码器的结构隐藏层通常使用神经网络。
SDNE： 是经典的基于自编码器的网络嵌入方法。受SDNE的启发，无监督的特性和出色的表现，出现了应用于动态网络的DynGEM算法。GynGEM： 保留上一时刻的嵌入信息，下一时刻将利用这些信息，使下一时刻的嵌入模型可以直接继承上一时刻训练的模型的参数。值得注意的是，网络节点和边的数量会对嵌入模型的体系结构产生影响。该方法利用启发式信息，根据新的网络结构调整SDNE的整体结构，便于应用于新的网络。简化后的DynGEM模型如下图所示。

dyngraph2vec： 该方法认为GyGEM以及其他动态图嵌入方法仅仅考虑了上一时刻的图信息，忽略了丰富的历史结构信息。因此，dyngraph2vec考虑了前L时刻的网络信息进行学习。具体来说，该算法将时间有序的网络序列表示为语料库，利用RNN对历史信息进行编码进行半监督学习，并给出一个简单的自编码器更新规则。然而，dyngraph2vec使用了三种模型来嵌入历史信息，这导致了高度的复杂性。

基于神经网络的图嵌入

近年来，神经网络在网络嵌入领域引起了研究者们的广泛关注。基于神经网络的网络嵌入模型凭借其强大的表达能力取得了显著的效果。

DyRep：归纳深度表示学习框架分解为网络上的两个动态过程:关联过程和通信过程。前者处理网络拓扑的变化，后者处理网络动态变化。该模型通过定义事件来统一表示上述两个过程的变化。此外，DyRep将节点表示的变化视为上述两个进程之间的一个中间桥梁，以便根据新的事件不断更新节点表示。当事件发生时，节点的新表示形式由与事件相关的邻居节点聚合。此外，以动态聚合节点邻居为目标的模型采用了注意机制。
DySAT构造节点表示，将自注意机制整合到两个方面:结构邻居和时间动力学。通过动态自注意体系结构，它提供了节点的动态表示，可以捕获结构属性和时间演化模式。

循环神经网络(RNN)通常用于连续序列学习，通过捕捉时间序列的动态信息。在RNN的启发下，一些动态网络嵌入方法将RNN和其他神经网络结合起来学习节点表示。
EvolveGCN包含两个组件，RNN组件和图卷积网络(GCN)组件。RNN模型在下一个时间步中更新权值矩阵，GCN模型使用更新后的权值矩阵和当前节点嵌入矩阵更新后续节点嵌入矩阵。EvolveGCN关注的是GCN参数在每个时间步的演化，而不是节点表示，其模型具有自适应和灵活性，能够保持网络动态。

其他嵌入方法

DynamicTriad[64]，它保存了网络的结构信息，并将网络的时变特性融入到嵌入向量中。通过对三角闭合问题的分析，设计了一种动态网络的演化来描述网络的变化。为了表达一些快速变化的节点，反映结构的演化，采用三角形闭合问题来反映网络的动态。DynamicTriad构造了损失函数来预测三角形的变化。
DGNN更详细地考察了边缘连接的作用时间与其对网络嵌入的影响之间的关系。它假设当新的边出现时，边的两端和端点的一阶邻居都有很高的概率发生改变。

数据集

本节介绍动态网络嵌入中常用的数据集，可用于动态网络分析。对于每个数据集，对不同的模型采用特定的预处理方法。动态网络包括合成动态图和真实动态图。我们总结了几个常用的动态数据集如下:

  • Academic 它是由51060位研究者作为顶点，794552位共同作者作为边缘组成的学术网络。快照的大小为2年。在每个快照中，如果存在两个有科研合作项目的用户，就会在两个用户之间产生一个边。每个研究人员都有其特定的社区标签。
  • Autonomous Systems(AS) 该数据集是一个通信网络，每条边记录用户之间的成功通信信息，包含了1997年至2000年733个用户的通信记录，可按年划分为快照。
  • CollegeMsg 该数据集包括加州大学在线社交网络上发送的私人信息。用户可以根据配置文件信息发起与网络中其他用户的对话。边(u,v,t)表示用户u在t时刻向用户v发送消息，193天内记录1899个节点和59835条边的信息，这些边被分成3个快照。
  • DBLP 这是一个学术网站，统计发表两篇以上论文的作者数量。从2001年到2016年，该数据集恢复了317080个作者和1049866条边之间的信息，每个作者拥有自己的研究领域标签。
  • Email-Eu-core 该数据集是由欧洲一家大型研究机构e-mail network生成的。它反映了研究所成员之间匿名的电子邮件交流。有42个研究所，所有成员都包括在内。电子邮件仅仅代表了组织核心成员之间频繁的日常交流。邮件总数为61046封，时间跨度为803天。
  • ENRON 该数据集包含了安然公司(Enron Inc.) 1999年1月至2002年7月期间员工之间的电子邮件.
  • Epinions Epinions是一个商品评论网站，用户可以在这里对各种商品发表评论和意见。此外，用户自身可以建立信任关系，这是用户之间寻求建议的基础。该数据集是一个真实世界的动态网络，数据具有16个不同的时间戳.
  • F2f-Resistance 该数据集包含62个子图、451个节点和3,126,993条边提取的网络。平均时间长度每个子图2290秒，可以分为4个快照。
  • HEP-TH 该数据集记录了1993年1月至2003年4月学术会议上作者之间的合作情况。
  • Infectious 这个网络记录了参观者在展览中的互动。每个边缘的形成需要参观者之间至少20秒的面对面交流，并有准确的时间戳。它只存储经常交流的访问者。
  • Internet 这就是互联网自治系统的连接网络。节点表示自治系统，边表示自治系统之间的连接。多个边可以连接两个节点，每个节点在时间上代表一个单独的连接。用连接的时间点标注边.
  • Loan 该网络的结构类似于ppdai3提供的移动数据集，其中包含了超过13个月的20万名注册拍拍用户的1603712条通话记录。快照大小设置为1个月。
  • Mobile 这是一个由中国电信提供的200多万条通话记录组成的移动网络。在15天内，保留了340751个节点和2200203条边。将用户视为顶点，将快照定义为连续的一天，且没有重叠。在每个快照中，如果一个用户呼叫另一个用户，用户顶点之间的连接将被记录在网络中。
  • Synthetic Data (SYD) 合成动态图通常采用随机块模型[56]生成。通过设置块间连通概率和块内连通概率生成合成数据快照。
  • Twitter 该数据集包含截至2015年5月31日的1375个Twitter用户的基本信息，以及所有注册用户的博客信息。用户成功的交互信息生成了378万条具有一定时间戳的边。
  • Wikipedia 它是一个来自维基百科的数据集，每条边代表用户之间有特定时间标签的通信。时间跨度为2320天，其中注册用户数为1140149，连接数为7833140

节点分类

节点分类的目的是将每个节点划分成相应的类别，便于下游应用。节点分类包括多标签分类和多类分类。网络嵌入采用的节点分类过程一般分为四个步骤。首先，应用网络嵌入算法推导出网络的嵌入表示。其次，将包含标签信息的数据集划分为训练集和测试集;然后，在训练集上训练分类器。最后，对测试集进行节点分类。选择的分类器包括liblineal、最近邻分类器、支持向量机等。通常使用Micro-F1和Macro-F1作为节点分类任务的评价指标，综合了分类模型的正确率和召回率。

链路预测

链路预测任务的目标是预测两个节点之间是否存在边，可以评估嵌入方法在保持显式网络拓扑结构方面的性能。动态网络嵌入中采用的链路预测分为三个步骤。首先，学习节点的嵌入表示。然后，将数据集划分为训练集和测试集。对于训练集和测试集中的任意一对节点，生成一个标签，记录这两个节点是否有连接。最后，在测试集上进行链接预测实验。AUC和MAP测度作为评价指标。MAP的目标是测量模型的召回率和精度，可以反映整体表现。与AUC相比，MAP考虑的是返回项的性能排名。

节点聚类

节点聚类是一种无监督任务，其目标是将一个网络划分为多个簇，属于同一簇的节点具有更多的相似性。与节点分类类似，节点聚类是为了学习节点的类别信息。在通过嵌入算法得到节点表示后，实际采用了一些典型的聚类方法进行节点聚类，如K-means。通常使用归一化互信息(NMI)评价网络嵌入算法的聚类性能。(聚类区别于分类，聚类仅仅知道分为了几类，并不知道每个类别的label信息，而分类知道每个类别的label信息。)

网络可视化

节点的嵌入表示被用作可视化工具的输入，如t-SNE[32]。通过嵌入算法生成节点表示法后，t-SNE首先将d维节点表示法映射到二维空间中，可视化结果显示在二维空间中。网络可视化是对带有标签的数据集进行的。在二维空间中，标签不同的节点用不同的颜色进行标记。由于网络嵌入保留了网络的固有结构，可视化结果直接反映了在二维空间中同一簇内的节点具有更多的相似特征。

未来研究方向

由于网络是随时间演化的，上述模型仅局限于两种模式，即快照模式和连续时间模式，难以适应更复杂的情况。因此，本部分讨论了未来工作的几个有前途的研究方向。

动态嵌入模型

考虑到网络的实时演化，实现网络嵌入执行的工作很少。例如，如果在网络中添加或删除一些边，现有的方法需要对原有的网络进行重新处理，然后进行网络嵌入，耗时较长。为了在网络状态变化时有效地更新嵌入表示，动态嵌入模型是非常理想的。

大规模网络的动态网络嵌入

对于网络嵌入，Tang等人[48]提出了LINE来分析大规模的复杂网络。但LINE不适用于动态网络。由于在动态网络中进行网络演化的复杂性，现有的动态网络嵌入方法不能适用于更复杂的真实网络。根据上述调查，动态网络可以表示为快照或时间戳标记的连续时间网络。快照或时间戳的数量越多，网络演化的复杂性就越高。因此，可以从降低网络演化的复杂性或改进嵌入模型两方面来提高动态网络嵌入的效率。综上所述，大规模动态网络嵌入是一个难以实现的研究难点。

异构网络中的动态网络嵌入

虽然目前的研究主要集中在动态网络嵌入上，但现有的方法主要是基于单一类型节点和边缘的同构网络。然而，许多现实世界的网络都是异构网络，其中存在几种类型的节点和边。异构网络的动态嵌入需要考虑网络随时间变化的节点类型和边缘类型。此外，如果从原始的异构网络中提取结构和性质，则在网络演化的过程中要尽可能地保留所提取的结构和性质。探索异构网络中的动态网络嵌入是一个有趣的研究方向。

基于属性网络的动态网络嵌入

属性网络的动态嵌入强调网络结构和节点属性随网络演化而提供的信息。

面向任务的动态网络嵌入

向任务的网络嵌入方法只关注一个任务，从而可以提取与任务相关的一些额外信息来训练嵌入模型。面向任务的嵌入模型比通用嵌入模型对目标任务更有效。然而，在动态网络嵌入中，还没有针对单个网络挖掘任务的相关工作。

更多的嵌入空间

一般情况下，目标空间嵌入越深，时间和空间复杂度越高。但另一方面，将网络嵌入到更多的空间中，也为保留网络结构和网络提供了更多的可能性。因此，探索不同的嵌入空间是动态网络嵌入的一个有趣且具有挑战性的研究方向
